#+TITLE: wiktionary-bro changelog

* [1.1.0] - 2025-12-09
- Smart link navigation: Wiktionary links open in wiktionary-bro buffers (RET)
- External links (Wikipedia, etc.) open in browser
- Multi-language support with customizable ~wiktionary-bro-language~
- Language switching in buffers with =C-c C-l= key to re-render entries
- Language picker shows only available languages for the current entry (from API langlinks)
- Internal links now respect the current language setting

* [1.0.0] - 2025-12-07
- ASCII table rendering with colspan/rowspan support
- Pronunciation audio playback via ffplay
- Buttercup tests and CI

* [0.1.0] - 2022-12-24
- Initial release
